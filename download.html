Upgrade required: legacy wallets and nodes no longer work.

The Salvium hard fork completed at block 465,000 on 13 April 2026. Legacy wallets and any CLI build older than v1.1.x can no longer sync with the network.
Please upgrade to the Salvium GUI Wallet v1.1.1 or the Salvium CLI v1.1.1b below. This release brings Salvium Assets, Carrot & SPARC support, and the new treasury-based emission model.
